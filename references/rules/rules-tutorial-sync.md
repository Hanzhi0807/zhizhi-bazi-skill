## 一、暗拱虚神（半合的隐藏力量）

> 📖 进阶教程 Page 37, 84, 114-115

### 核心规则

三合局只出现长生位和墓库位（缺帝旺位）时，会**暗拱**出虚神（帝旺位），形成完整的三合局。暗拱的虚神一旦被大运流年"填实"（透干或地支到位），潜在力量就会爆发。

### 查表

| 半合组合 | 暗拱虚神 | 完整三合局 |
|:---|:---|:---|
| 亥 + 未（缺卯） | **卯** | 亥卯未木局 |
| 申 + 辰（缺子） | **子** | 申子辰水局 |
| 寅 + 戌（缺午） | **午** | 寅午戌火局 |
| 巳 + 丑（缺酉） | **酉** | 巳酉丑金局 |

### 判断要点

1. **暗拱≠半合**：半合是长生+帝旺或帝旺+墓库(有帝旺参与)，暗拱是长生+墓库(缺帝旺)。两者不同——半合已有合力，暗拱是虚神待填。
2. **暗拱的虚神是潜在的**：原局不发挥作用，但大运流年填实(透干或地支到)则爆发。
3. **大器晚成的信号**：原局有暗拱者，格局力量潜伏，待填实大运方显——是"大器晚成"的重要判断线索。

### 典型案例

曾国藩命造年月亥未暗拱卯木(印局)，到 46 岁走甲午运甲木透出，暗拱木局填实，木火助身，成就"同治中兴"。

### 何时使用

- 流转分析(Step 5)地支流转时，检查是否有"长生+墓库缺帝旺"的暗拱组合
- 大运流年分析(Step 10)时，大运填实暗拱虚神=格局力量爆发点
- 历史校准时，若命主有大器晚成特征，检查原局暗拱

## 二、母慈灭子（反局）——印过旺的例外

> 📖 进阶教程 Page 43-44；《滴天髓》"反局"篇

### 核心规则

印星过旺时，**不但不生身，反而"母多灭子"**（如土重金埋、水多木漂、火多土焦）。此时喜用不是跟着印走，反而要**扶日主、制印星**。这与常规"印旺则身强"的推理相反。

### 判断条件

1. 印星在命局中**数量过多**(3 个以上)或**当令而重**(月令印星 + 天干透印)
2. 日主**无根或根弱**(无禄、无帝旺、无长生位)
3. 无比劫帮身(比劫能分担印之压力)

三者同时满足 → 母慈灭子，印星反成负担，用神改为比劫(帮身抗印)或财星(坏印存身)，**不能取印为用**。

### 典型案例

四柱皆土、辛金日主无根无比劫——土重金埋。走庚申运(比劫帮身)助起日元方得机遇。

### 何时使用

- 旺衰判断(Step 3)时，若印星多而日主无根，**不能简单把印计入"日主同方"助身**，要考虑母慈灭子
- 用神选取(Step 4)时，若判母慈灭子，用神从"印"反转为"比劫/财"
- Anti-Pattern 自检：是否犯了"印多=身强"的惯性思维？

## 三、根与印的本质区别

> 📖 进阶教程 Page 44, 46

### 核心规则

根(地支同类主气，即比劫之根)和印星(生我之物)虽同属"日主同方"，但**不可等同**：

| 维度 | 根(比劫之根) | 印(生我之物) |
|:---|:---|:---|
| 代表六亲 | 兄弟同辈 | 父母长辈 |
| 力量性质 | 自身力量、独立自主 | 庇护、依赖、被动接受 |
| 无根有印 | —— | 依赖性强，难自立 |
| 有根无印 | 可自立 | —— |
| 印过旺无根 | 母慈灭子(见上) | —— |

### 关键推论

1. **无根的日主，印再多也旺不了**——这是母慈灭子的底层逻辑。
2. **有根无印尚可自立，无根有印则依赖性强**——性格与人生轨迹不同。
3. **官杀是"两头蛇"**：官杀本克我，但天干透印时，官杀生印、印再生身，官杀起的不是"克"而是"生"的功能。

### 何时使用

- 旺衰判断(Step 3)时，根和印分开计权——根的权重高于印(根是自身力量，印是外部庇护)
- 用神选取时，"扶身"优先取比劫(根)，其次取印——因为比劫是自身力量，印是外部依赖
- 性格分析时，有根无印=独立自主，无根有印=依赖性强

## 四、墓与库的区分

> 📖 进阶教程 Page 100-104

### 核心规则

辰戌丑未四土既是墓也是库，区分如下：

| 区分 | 透干情况 | 含义 | 操作 |
|:---|:---|:---|:---|
| **库** | 墓库所藏人元**透天干** | 可用、有力量 | 正常使用，待时而发 |
| **墓** | 墓库所藏人元**不透** | 死藏、死气沉沉 | 需冲开方可用 |

> 📖 古诀："少年不发墓中人"——墓神不透则死气沉沉，不能堪用。

### 逢墓与入墓的细分

| 类型 | 含义 | 出墓难度 |
|:---|:---|:---|
| **逢墓** | 十神是墓库的**主气**(如戊土是戌的主气) | 浅埋，易出 |
| **入墓** | 十神是墓库的**附属之气**(如乙木是戌的墓神) | 深埋，难出 |

### 开库条件

墓库**逢冲**才能"开库"取用：
- 辰戌冲、丑未冲——开土库
- 原局有墓库不透，大运流年冲墓库=开库，力量爆发
- 财星入库无冲=难发；财星入库逢冲=暴发

### 典型案例

慈禧太后七杀库被未冲开，独掌大权 47 年。

### 何时使用

- 流转分析(Step 5)时，地支有辰戌丑未需区分墓/库——透干为库、不透为墓
- 大运流年(Step 10)时，冲墓库=开库，是 timing 触发点
- 财官分析时，财官入库需冲方发，无冲则潜伏

## 五、反生反克原理

> 📖 基础教程 Page 112-113

### 核心规则

五行除正常生克外，在**某五行过旺或过衰**时，正常的生克方向会逆转，称为"反生反克"。

### 五组反生（生扶太过反遭害）

| 正常 | 反生（生扶太过） |
|:---|:---|
| 金赖土生 | **土多金埋** |
| 土赖火生 | **火多土焦** |
| 火赖木生 | **木多火塞** |
| 木赖水生 | **水多木漂** |
| 水赖金生 | **金多水浊** |

### 五组我生反衰（泄太过）

| 正常 | 反衰（我生太过） |
|:---|:---|
| 金能生水 | 水多金沉 |
| 水能生木 | 木多水缩 |
| 木能生火 | 火多木焚 |
| 火能生土 | 土多火晦 |
| 土能生金 | 金多土弱 |

### 五组反克（克者太旺被反克）

| 正常 | 反克（被克者旺） |
|:---|:---|
| 金能克木 | **木旺金缺** |
| 木能克土 | **土重木折** |
| 土能克水 | **水多土流** |
| 水能克火 | **火多水灼** |
| 火能克金 | **金多火熄** |

### 五组衰极不堪再制

| 衰极 | 再制 |
|:---|:---|
| 金衰遇火 | 必见销熔 |
| 火弱逢水 | 必藏熄灭 |
| 木弱逢金 | 必被砍伐 |
| 水弱逢土 | 必被淤塞 |
| 土弱逢木 | 必被倾陷 |

### 五组强旺需泄

| 强旺 | 需泄 |
|:---|:---|
| 强金得水 | 方挫其锋 |
| 强水得木 | 方缓其势 |
| 强木得火 | 方泄其秀 |
| 强火得土 | 方止其焰 |
| 强土得金 | 方制其壅 |

### 何时使用

- 用神选取(Step 4)时，若某五行过旺，**不能简单取生扶或克制的常规用神**——要考虑反生反克
  - 例：土多金埋时，印星(土)反成负担，不能取印为用，要取比劫(金)帮身或财(木)坏印
  - 例：木旺金缺时，官杀(金)克不了木反被缺，要取食伤(水)泄木或火通关
- 旺衰判断(Step 3)时，印过旺要考虑"反生"——印多反埋身
- 调候分析(Step 2)时，某五行过旺(如夏火过旺)要考虑"火多土焦"——调候用神可能反受害

## 六、大运干支五类型（盖头/截脚等）

> 📖 基础教程 Page 200-203

### 核心规则

六十甲子按干支生克关系分为五种类型，各有不同吉凶特征：

| 类型 | 干支关系 | 吉凶特征 | 喜忌判断 |
|:---|:---|:---|:---|
| **干支同气** | 干支同类(如甲寅、丙午、庚申) | 最强，吉凶最显著 | 喜则大吉，忌则大凶 |
| **上生下** | 天干生地支(如甲午、丙戌) | 地支得生助，吉凶多在**后五年** | 喜神在地支则吉；忌神在地支则后五年凶 |
| **下生上** | 地支生天干(如甲子、丙寅) | 天干得支持，吉凶多在**前五年** | 喜神在天干则前五年吉；忌神在天干则前五年凶 |
| **上克下（盖头）** | 天干克地支(如甲申、丙申) | 喜神在地支则被盖头，**吉凶参半** | 喜神在地支=被盖、打折；忌神在地支=被盖、减凶 |
| **下克上（截脚）** | 地支克天干(如甲申?——申金克甲木) | 喜神在天干则被截脚，**十年全空** | 喜神在天干=被截、十年皆空；忌神在天干=被截、减凶 |

### 关键结论

> **盖头与截脚比较**：喜神在地支遇盖头只是吉凶参半(地支之力打折但天干仍作用)；喜神在天干遇截脚则是十年全空(天干被地支截断，力量全无)。

### 大运重地支

教程指出：**大运重地支**，因为地支主方向(寅卯辰=东方等)。观察天干运时须联系地支，观察地支运时可舍弃天干。

### 何时使用

- 大运分析(Step 10)时，每步大运先分类型——干支同气/上生下/下生上/盖头/截脚
- 大运吉凶分前后五年时，按"上生下后五年、下生上前五年"判断
- 喜神在天干遇截脚=此运十年喜神全废，需特别注意

## 七、官杀组合五型 + 三大领域

> 📖 基础教程 Page 253-255

### 官杀格五型成功组合

| 类型 | 格局条件 | 用神 | 象义 |
|:---|:---|:---|:---|
| **官印相生** | 正官格身弱，印化官生身 | 印星 | 清贵，适合文职/学术 |
| **财生官** | 正官格身强，财滋官 | 财星 | 富贵双全，工商仕途皆宜 |
| **杀印相生** | 七杀格身弱，印化杀生身 | 印星 | 文武兼备，重名轻利 |
| **财滋弱杀** | 七杀格身强杀弱，财生杀 | 财星 | 以杀为威，武贵 |
| **食神制杀** | 七杀格身杀两停，食神制杀 | 食神 | 武贵，刚烈有魄力 |

### 三大领域映射

| 十神关系 | 对应领域 | 人生方向 |
|:---|:---|:---|
| 日主—财星 | 经济领域 | 财富、商业、物质资源 |
| 日主—官杀 | 政治领域 | 权力、地位、管理、仕途 |
| 日主—印星—食伤 | 知识/精神领域 | 学问、技艺、精神追求、文化 |

### 何时使用

- 格局判断(Step 4)时，官杀格按此五型归类——快速定位用神与象义
- 综合建议(Step 13)时，按三大领域判断命主人生重心——财官印食伤哪个领域是主线
- 校准时，按命主实际从事领域反推格局是否匹配(如命主是学者却财官双美，需重新审视)

## 八、六亲星宫交叉分析法

> 📖 基础教程 Page 228-231, 271

### 核心规则

六亲判断需"星"和"宫"**两条线交叉**：

### 星(十神)线

- 看某六亲星是否**健旺**(有根、不被克破)
- 看是**喜用**还是**忌神**
  - 星为喜用且健旺 → 该六亲助力大
  - 星为忌神 → 该六亲反成负担
- 六亲星映射：正印=母、偏财=父、正官=夫(女命)、正财=妻(男命)、食神伤官=子女(女命)、七杀=子(女命)、比肩劫财=兄弟

### 宫(四柱位置)线

| 宫位 | 对应六亲 |
|:---|:---|
| 年柱 | 祖上、父母(尤其祖辈) |
| 月柱 | 父母、兄弟 |
| 日支 | 配偶 |
| 时柱 | 子女 |

- 看宫位是否遭到**刑冲破害**
- 宫位安稳则该六亲稳，宫位被冲则该六亲不稳

### 交叉分析

星出现在哪个宫位有额外含义：
- **印星(母)出现在日支夫妻宫** → 婚后仍与母亲同住，或配偶有母辈特质
- **财星(妻)出现在月柱父母宫** → 妻与父母关系密切，或早婚
- **食伤(子女)出现在年柱祖上宫** → 子女与祖辈缘深，或晚年得子归祖

### 六亲推导逻辑

- "子由母出"：印生身，印为母；食伤生财，食伤为财之母
- "夫妻克合"：阴阳相克相合，男以正财为妻(我克者)，女以正官为夫(克我者)
- 可层层推出 13-15 个近亲

### 何时使用

- 六亲分析时，**不能只看星或只看宫**——必须两条线交叉
- 校准时，六亲事件(父母亡故、婚配、得子)按星+宫+大运流年三层推断
- 综合建议时，按星宫交叉判断六亲缘分深浅

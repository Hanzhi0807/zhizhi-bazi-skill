## 一、病药说

> 📖 原文卷一"病药说类"

### 核心思想

```
有病方为贵，无伤不是奇；格中如去病，财禄两相随。
```

**病** = 八字中原有所害之神（命局的矛盾/失衡点）
**药** = 能去除病患之字（解决矛盾的关键元素）

### 找病的方法

张楠提出"从重者论"的下手法：

1. 先看日干
2. 次看月令，找出月令中最旺的五行
3. 再看年、时是否也有同五行
4. 将同一五行的干支集中看：是为病？还是非病？

> 📖 原文："如八字中先看了日干，次看月令……宜将以上各火做一处看，或为病，或非病。"

### 常见病药对应

| 病症 | 病理 | 药方 | 说明 |
|------|------|------|------|
| 杀重身轻 | 七杀过旺克身 | 食伤制杀 | 水日干四柱纯土 |
| 土厚埋金 | 印重压身 | 木疏土 | 金日干土过重 |
| 财多身弱 | 财旺身不胜 | 比劫帮身/印生身 | 富屋贫人 |
| 比肩太重 | 比劫夺财 | 官杀制比劫 | 火日干比劫林立 |
| 财被劫夺 | 财弱比劫旺 | 官杀护财 | 用财见比肩为病→官杀为药 |
| 食神被夺 | 枭神夺食 | 财制枭 | 用食伤以印为病→财为药 |
| 母慈灭子 | 印重克食伤 | 财破印 | 印过旺制食伤 |

### 病药与中和

- "纯然不旺不弱，原财官印俱无损伤，日干之气又得中和，并无起发可观"→ **平常人也**
- "病重而得药，大富大贵之人也；病轻而得药，略富略贵之人也；无病而无药，不富不贵之人也"

> 📖 原文："若一于中和，则安得探其消息，而论其休咎也？"

## 二、辟谬说

> 📖 原文"五星谬说""子平诸格谬说"等

### 张楠辟谬的核心批判

| 谬说 | 张楠的批判 | 现代启示 |
|------|-----------|---------|
| 盲目追求中和 | "以中和究人之造化，十无一二有验" | 有病有药方为贵 |
| 神煞泛滥 | 五星神煞之说不可靠 | 神煞只作辅助，不作主要依据 |
| 死套格局 | 不必每个八字都强套一个格局名 | 格局是工具不是目的 |
| 财官至上 | 不是所有命局都以财官为贵 | 要看日主能否承受 |
| 星辰决定论 | "星辰无关格局"（与子平真诠同调） | 十神为主，星辰为辅 |
| 纳音万灵 | "在愚见论之，妄诞甚矣" | 纳音为旁证，不代核心判断 |

### 辟谬的方法论

- **"不从古人之说"**：不盲从古人每一句话
- **"以理断之"**：以五行生克之理为核心判断标准
- **"以验为据"**：用实际命例反验证理论

## 三、雕枯旺弱四病

> 📖 原文"雕枯旺弱四病说类"

### 雕（损伤）

日主或格局用神被克，如：
- 木被金伤 → 需水通关（金生水，水生木）
- 金被火熔 → 需土通关（火生土，土生金）

### 枯（干枯）

五行缺乏生机，如：
- 夏木无水 → 枯槁
- 冬金无火 → 寒冻
- **解法**：调候为先

### 旺（过强）

某五行过旺：
- 旺者宜泄（食伤泄秀）或宜克（官杀制之）
- 不可强克，宜顺势疏导

### 弱（过弱）

日主过弱：
- 弱者宜生（印绶相生）
- 弱极不宜扶，可能为从格 → 从则相济

## 使用边界

- 病药说是一种有用的思维框架，但"病"的认定存在主观性
- 张楠对神煞的批判较为极端，实际分析中神煞可作为低优先级的辅助参考
- 辟谬精神的核心是"以理断之"——不是否定所有古籍，是否定不经论证的照搬

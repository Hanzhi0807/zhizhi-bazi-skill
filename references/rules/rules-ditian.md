## 一、天干性情（定性基础）

> 📖 原文第七章"天干"

### 十天干本质

| 日主 | 本质特征 | 关键提示 |
|------|---------|---------|
| 甲木 | 参天雄壮，脱胎要火 | 春不容金，秋不容土；火炽乘龙（辰），水宕骑虎（寅） |
| 乙木 | 虽柔，刲羊解牛 | 怀丁抱丙，跨凤（酉）乘猴（申）；藤萝系甲，可春可秋 |
| 丙火 | 猛烈，欺霜侮雪 | 能煅庚金，逢辛反怯；土众成慈，水猖显节 |
| 丁火 | 柔中，内性昭融 | 旺而不烈，衰而不穷；如有嫡母（甲），可秋可冬 |
| 戊土 | 固重，既中且正 | 水润物生，火燥物病；若在艮坤（寅申），怕冲宜静 |
| 己土 | 卑湿，中正蓄藏 | 不愁木盛，不畏水狂；火少火晦，金多金光 |
| 庚金 | 带煞，刚健为最 | 得水而清，得火而锐；土润则生，土干则脆 |
| 辛金 | 软弱，温润而清 | 畏土之叠，乐水之盈；热则喜母（己土），寒则喜丁 |
| 壬水 | 通河，刚中之德 | 周流不滞；通根透癸，冲天奔地；化则有情，从则相济 |
| 癸水 | 至弱，达于天津 | 得龙（辰）而运，功化斯神；不愁火土，不论庚辛 |

### 五阳从气不从势，五阴从势无情义

- **阳干（甲丙戊庚壬）**：有阳刚之气，不轻易从财杀之势。即使身弱，若有微根或生气，仍可自立
- **阴干（乙丁己辛癸）**：顺势而为，势力强大时倾向于从。木盛则从木，火盛则从火

> 📖 原文："五阳得阳之气，即能成乎阳刚之势，不畏财杀之势；五阴得阴之气，即能成乎阴顺之义"

## 二、旺衰判断规则

> 📖 原文第十七章"衰旺"及其它相关章节

### 判断五要素

按优先级排列：

| 优先级 | 要素 | 判断方法 |
|--------|------|---------|
| 1 | **月令** | 日主是否得月令生旺之气？此为"司令" |
| 2 | **根气** | 地支是否有日主之根（禄、旺、长生、库、余气）？ |
| 3 | **生扶** | 天干是否有印星、比劫生扶？ |
| 4 | **克泄耗** | 财官食伤是否过重，压制日主？ |
| 5 | **气势流通** | 全局气势是流向日主还是流走？ |

### 旺衰等级

| 等级 | 特征 | 倾向 |
|------|------|------|
| **身强** | 得令+得地+得助，三项占二 | 喜克泄耗（财官食伤） |
| **身旺** | 得令或得地，但克泄耗亦不弱 | 看气势流通方向 |
| **中和** | 生扶与克泄耗大致均衡 | 最贵之局，随运而变 |
| **身弱** | 失令+无强根，但有印比 | 喜生扶（印比） |
| **身衰** | 失令+无根+无生扶 | 可能从势，不可强扶 |
| **假旺/假弱** | 表面旺但根气受损，或表面弱但气势暗藏 | 需仔细辨别流通 |

### 关键诊断规则

1. **旺者冲衰衰者拔**：旺支冲动衰支，衰支被拔除，不能为用
2. **衰神冲旺旺神发**：衰支冲动旺支，旺支被激发，反而更旺
3. **地生天者，天衰怕冲**：日主坐长生之地，若日主衰弱，最怕冲克
4. **天合地者，地旺喜静**：日主坐下与天干相合，若坐下旺相，宜静不宜冲

> 📖 原文："旺者冲衰衰者拔，衰神冲旺旺神发"（第九章）

## 三、气势流通规则

> 📖 原文第十九章"源流"、第二十章"通关"

### 源流判断

看全局五行是顺生还是阻滞：

- **流向我**（印→日主、比劫助身）：身得补益
- **流向他**（日主→食伤→财→官→印→日主）：循环良好，贵格
- **阻滞**：某环节被克破，气势中断，为病

### 判断问题

| 问题类型 | 表现 | 解法 |
|----------|------|------|
| 身强无泄 | 比劫林立，无食伤财官 | 需行食伤财运 |
| 身弱无扶 | 财官过重，无印比 | 需行印比运 |
| 火炎土燥 | 火土过旺，无水润泽 | 需壬癸水调候 |
| 水冷金寒 | 金水过旺，无火温暖 | 需丙丁火调候 |
| 金多水浊 | 金多水少，水被浊 | 水须有根方澄清 |
| 木多火窒 | 木多火塞，火不从 | 需金制木以救火 |

### 通关原理

> 📖 原文第二十章"通关"

两神相战，取通关之神调和：
- **金木相战** → 水通关（金生水，水生木）
- **木土相战** → 火通关（木生火，火生土）
- **土水相战** → 金通关（土生金，金生水）
- **水火相战** → 木通关（水生木，木生火）
- **火金相战** → 土通关（火生土，土生金）

> 📖 原文："关内有织女，关外有牛郎，此关若通也，相将入洞房"

## 五、从化判断规则

> 📖 原文六亲论"从象""化象""假从""假化"

### 真从条件

1. 日主失令、无根、无生扶
2. 全局某五行独旺，形成不可抗拒之势
3. 不可有任何帮扶日主的元素

### 假从条件

1. 日主极弱但有微根（如一点印星藏支）
2. 大部分从势，但有一二处不纯
3. 岁运去其病则发，助其病则败

### 真化条件

1. 日主与月干/时干相合
2. 月令为化神之旺地
3. 支成化神之局
4. 无克破化神之字

### 关键规则

- **"化则有情，从则相济"**：从格富贵需岁运扶起所从之神
- **假从假化**：运势扶起即为真，运势背反即败露
- **不可轻言从化**：大多数命局是正常格局，不是从格

## 七、形象与成象（形象视角）

> 📖 原文："两气合而成象，象不可破也。"（《滴天髓》"象"章）

### 形象视角的定位

形象是与强弱、调候、格局并列的**第四个分析视角**，专治"气势偏于一端"的命局。常规格局用月令取格、扶抑定用，但当一个八字的两行或一行之气已成压倒之势，再用扶抑逻辑（身弱扶、身旺抑）就会出错——此时必须转用形象视角。

> 陆致极《进阶教程》明确：形象视角"是一个重要的视角，有必要予以充分重视。有了这个视角，命局中的不少组合就一目了然了"。以往命书少把"形象"单列，《滴天髓》虽涉及但言过于简。

### 形象成立与否的判断

| 检查项 | 成象特征 |
|--------|---------|
| **两行成象** | 全局基本只有两行，且两行相生（木火通明、金水相生、水木清华、火土相生、土金相生），气势专一 |
| **一行专旺** | 全局一行独旺，成方成局（曲直、炎上、稼穑、从革、润下），或虽未全成方局但势已偏极 |
| **象已成形** | 上述气势通过天干透出、地支会局或合化落实到具体干支，非虚浮 |
| **不可破** | 成象之后，岁运见克象之五行反为祸，见顺象之五行则发 |

### 核心原则：象不可破

```
象成 → 顺其势，不可逆克
  ├─ 木火通明：顺木火，忌水破火、金克木
  ├─ 金水相生：顺金水，忌土克水、火克金
  ├─ 火土相生：顺火土，忌水破火、木克土
  ├─ 水木清华：顺水木，忌土克水、金克木
  └─ 土金相生：顺土金，忌木克土、火克金

专旺格（曲直/炎上/稼穑/从革/润下）：
  └─ 顺其旺势，忌克、忌泄过甚；见克则祸，见食伤泄秀反佳
```

> 📖 原文："象不可破也。"——一旦成象，逆象之五行即为忌神，纵是常规喜用亦不可用。

### 形象视角与普通从格的区别

| 维度 | 普通从格 | 形象（成象） |
|------|---------|------------|
| 适用 | 日主无根、气势偏于一端、不得不从 | 不论日主有无根，只要两行/一行气势专一成象 |
| 日主 | 通常身衰无依 | 可身旺可身弱，关键是"势"已成 |
| 处理 | 从所从之神，忌帮扶日主 | 顺象之势，忌破象之五行 |
| 典型 | 从财、从杀、从儿、从旺 | 木火通明、金水相生、专旺五格 |

### 形象视角的操作步骤

1. **先扫全局**：数五行分布，看是否只有两行或一行独旺。若五行分散（三行以上且有生有克），**不走形象视角**，回到常规格局分析。
2. **判断是否成象**：两行须相生专一、一行须成方局或势极。虚浮两行不算成象。
3. **定象之顺逆**：成象后，生扶象之五行为喜，克破象之五行为忌。
4. **校验岁运**：岁运顺象则发，破象则祸。尤其注意"象不可破"——常规扶抑喜用到此处可能反转。

### 典型命例（陆致极书）

- **朱熹命造**（甲木日主，地支寅午戌合火局，伤食旺势）：木火通明之象，火势炎炎，"象不可破"。晚年"庆元党禁"之祸，正发于癸巳大运、丙辰流年——癸水克丙、辰冲戌破火局，逆象之五行到位，"一时真是黑云压城"。
- **左宗棠命造**（丙火日主，年月金水、日时木火对峙）：具木火与金水对峙的形象特点，"水火不容"落实到个性偏激；配合阳刃驾煞格局，成就其"以凶制凶"的刚烈一生。

> ⚠️ 形象视角不取代格局判断，而是并列的第四视角。常规命局（五行分散、有生有克）用格局视角即可；只有气势偏于一端的命局才需要启用形象视角。两者可并存于同一命局（如左宗棠：格局上是阳刃驾煞，形象上是金水木火对峙）。

## 使用边界

- 《滴天髓》的核心精神是"不可执一"——以上规则是指导框架，非机械判据
- 旺衰判断必须结合月令、根气、生扶、克泄耗、流通五要素综合评估
- 从化判断需严格满足条件，不可轻易下结论
- 通关是"最高级的解法"——两神相战时先找通关，而非直接帮扶弱者
- 寒暖燥湿的判断优先级高于格局、低于节气事实

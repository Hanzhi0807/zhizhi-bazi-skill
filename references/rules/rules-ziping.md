## 一、用神总则

> 📖 原文第八章"论用神"

### 用神从月令取

**用神专属月令**，以日干配月令地支，生克不同而定格局：

### 善用神（顺用）

| 用神 | 喜 | 作用 |
|------|-----|------|
| **财** | 食神相生 | 食生财 |
| | 官星护财 | 官制劫护财 |
| **官** | 财星相生 | 财生官 |
| | 印星护官 | 印制伤护官 |
| **印** | 官煞相生 | 官煞生印 |
| | 劫财护印 | 劫制财护印 |
| **食神** | 身旺相生 | 身强食旺 |
| | 财星化食 | 食生财 |

### 不善用神（逆用）

| 用神 | 制化方式 | 作用 |
|------|---------|------|
| **七煞** | 食神制伏 | 食制煞为权 |
| | 忌财印资扶 | 财生煞、印护煞则凶 |
| **伤官** | 佩印制伏 | 印制伤存官 |
| | 生财化伤 | 伤生财则秀 |
| **阳刃** | 官煞制伏 | 官煞制刃 |
| | 忌官煞俱无 | 无制则凶 |
| **月劫** | 透官制伏 | 官制劫 |
| | 透食化劫 | 食生财化劫 |

> 📖 原文："财官印食，此用神之善而顺用之者也；煞伤劫刃，用神之不善而逆用之者也。当顺而顺，当逆而逆，配合得宜，皆为贵格。"

### 七煞格的三种取用（制、化、抗）

七煞为不善用神，逆用之道并非只有"食神制伏"一路。按《子平真诠》逆用精神并参以实战，七煞格有**三条成格路径**，依命局条件择一而用：

| 取用 | 机制 | 成格条件 | 用神 | 相神 |
|------|------|---------|------|------|
| **制煞** | 食神制伏七煞，求自身与七煞的平衡 | 食神透干或为地支主气，身煞两停 | 食神 | 七煞 |
| **化煞** | 印星化煞生身，转克为生 | 身弱煞强，食神不透或制不住，有印星通根 | 印星 | 七煞 |
| **抗煞（阳刃驾煞）** | 以阳刃助身，以身抗煞，"以凶制凶" | 身弱煞强，食神不透、印化不通，日干为阳干且地支有阳刃 | 阳刃（或刃位之五行） | 七煞 |

**择用次序（陆致极《进阶教程》左宗棠命例所示）：**

1. 先看能否**制煞**——食神透干有力则取食制，此为正路。
2. 食神不透或制不住，再看能否**化煞**——有印星通根能化煞生身则取印化。
3. 制、化皆行不通，方考虑**抗煞**——阳刃驾煞，以凶制凶，"未免过于激烈"，为七煞格中最刚烈一路。

> ⚠️ 三者不可并用。食制与印化同见，须看位置先后与力量对比：食在前印在后，食被印制则煞无制；印在前食在后，印化煞生身、食再去制余煞，反为清格。阳刃驾煞一旦成立，岁运最忌财生煞旺、冲克阳刃。

> 📖 依据：《子平真诠》"煞逢食制"为逆用正路；《滴天髓》"旺极宜泄不宜克"对应化煞；阳刃驾煞为实战中"身弱煞强、制化皆穷"时的变格。左宗棠命造（丙火日主、庚金七煞透、午火阳刃）即取阳刃驾煞成格。

## 二、用神成败救应

> 📖 原文第九章

### 成格条件

| 格局 | 成格条件 |
|------|---------|
| 正官格 | 逢财印，无刑冲破害 |
| 财格 | 财生官旺，或财逢食生而身强，或财透印而位置妥贴 |
| 印格 | 印轻逢煞，或官印双全，或身印两旺用食伤泄气 |
| 食神格 | 食神生财，或食带煞而无财 |
| 七煞格 | 身强七煞逢制 |
| 伤官格 | 伤官生财，或伤官佩印，或伤官带煞而无财 |
| 阳刃格 | 透官煞而露财印，不见伤官 |
| 建禄月劫 | 透官逢财印，或透财逢食伤，或透煞遇制伏 |

### 败格条件

| 格局 | 败格原因 |
|------|---------|
| 正官格 | 逢伤克刑冲 |
| 财格 | 财轻比重，或财透七煞 |
| 印格 | 印轻逢财，或身强印重而透煞 |
| 食神格 | 逢枭（偏印夺食），或生财露煞 |
| 七煞格 | 逢财无制（食神不透、印星不化、又无阳刃驾煞，三路皆绝） |
| 伤官格 | 非金水而见官，或生财带煞 |
| 阳刃格 | 无官煞 |
| 建禄月劫 | 无财官，透煞印 |

### 救应方法

| 败因 | 救应 |
|------|------|
| 官逢伤 | 透印制伤 |
| 官杂煞 | 合煞清官 |
| 刑冲破害 | 会合解之 |
| 财逢劫 | 透食化劫，或透官制劫 |
| 印逢财 | 劫财解之，或合财存印 |
| 食逢枭 | 就煞成格，或生财护食 |
| 煞逢食制印护 | 透财去印存食 |

> 📖 原文："成中有败，必是带忌；败中有成，全凭救应。"

## 二·补：透干变格速查表（月令本气不透时的格局判断）

> 📖 依据：《子平真诠》"用神专寻月令，以四柱配之"——月令定体，透干定用。

### 为什么需要这张表

**高频误判模式**：月令本气为某十神（如己土正印），判断者立刻锁定"正印格"，忽略天干透出的其他十神（如癸水伤官）。当月令本气**不透**时，透干的十神往往才是真正的用神，可另起格局（伤官佩印、食神生财、七杀格等）。

**判断顺序（必须按此顺序，不可跳过）：**

1. 先看月令本气是否透干。透则本格成立（如己土透 → 正印格）。
2. 若月令本气**不透**，逐个检查透干的十神——每一个透干十神都可能另起格局。
3. 月令本气格与透干变格**可并存**（月令定体、透干定用），但用神以透干变格为准。
4. 若多个透干十神都能成格，按"成格条件齐备度"择优为主格，其余为兼格。

### 透干变格速查

| 月令本气（不透） | 透干十神 | 可成变格 | 成格四条件（须全部满足） |
|:---|:---|:---|:---|
| 印（己/戊） | 伤官（癸/壬） | **伤官佩印** | ①伤官旺有根 ②身稍弱 ③印有根（月令当令即可） ④印制伤存秀 |
| 印（己/戊） | 食神 | 食神泄秀 / 食神佩印 | ①食神旺有根 ②身强或中和 ③印有根 ④印不夺食（非偏印） |
| 印（己/戊） | 七杀 | 杀印相生 | ①杀有力 ②印有根 ③身稍弱 ④无财坏印 |
| 印（己/戊） | 正官 | 官印相生 | ①官有力 ②印有根 ③无刑冲破害 ④无伤克官 |
| 印（己/戊） | 财 | 财坏印（败格） / 财透印位置妥贴（成格） | ①财与印位置不相碍 ②有劫护印 ③身能任财 |
| 食（甲/乙） | 财 | 食神生财 | ①食旺有根 ②财有根 ③身强或中和 ④无枭夺食 |
| 食（甲/乙） | 杀 | 食神制杀 | ①食有力 ②杀有力 ③身煞两停 ④无财化食生杀 |
| 食（甲/乙） | 印（偏） | 枭夺食（败格） | ——（此为败格信号，非成格） |
| 官（丙/丁） | 财 | 财生官 | ①财有根 ②官有根 ③无伤克官 ④无刑冲 |
| 官（丙/丁） | 印 | 官印相生 | ①官有力 ②印有根 ③无财坏印 ④无伤克官 |
| 官（丙/丁） | 伤 | 伤官见官（败格） / 金水伤官见官（调候特例） | ①金水伤官见官反秀 ②非金水则需印制伤救应 |
| 杀（庚/辛） | 食 | 食神制杀 | ①食有力 ②杀有力 ③身煞两停 ④无财化食 |
| 杀（庚/辛） | 印 | 杀印相生 | ①杀有力 ②印有根 ③身稍弱 ④无财坏印 |
| 杀（庚/辛） | 阳刃 | 阳刃驾杀 | ①日干为阳干 ②地支有阳刃 ③身弱煞强 ④制化皆穷 |
| 财（戊/己） | 食 | 食神生财 | ①食有力 ②财有根 ③身强 ④无比劫夺财 |
| 财（戊/己） | 官 | 财生官 | ①财有根 ②官有根 ③无伤克官 |

| 建禄/月劫 | 官 | 建禄用官 | ①官有力 ②无伤克 ③有财印相配 |
| 建禄/月劫 | 杀 | 建禄用杀 | ①杀有力 ②有食制或印化 ③身强 |
| 建禄/月劫 | 财 | 建禄用财 | ①财有根 ②有食伤生财 ③无比劫夺财 |
| 建禄/月劫 | 伤 | 伤官泄秀 / 伤官佩印 | ①伤官旺 ②身强 ③佩印则印有根 |

### 使用示例

**误判案例（本表所防）**：庚金日主，月令未（己土正印本气），月干透癸水伤官。月令本气己土**不透**，透干的是癸水伤官。查表第一行："印不透 + 伤官透 → 伤官佩印"。成格四条件：①癸水伤官有根（年支亥）②庚金身稍弱 ③己土印当令有根（未）④印制伤存秀。四条全满足 → **伤官佩印格**，不是正印格。

> ⚠️ **判断格局前必须查此表。** 如果月令本气不透而你没有查这张表，格局判断大概率会锚定在"月令本气格"而漏掉真正的透干变格。

## 三、用神格局高低

> 📖 原文第十二章

### 评判二要素：有情 + 有力

| 维度 | 含义 | 高格 | 低格 |
|------|------|------|------|
| **有情** | 用神与辅神配合得当 | 官透财印相生，互不相碍 | 印用七煞而身强印旺（无情） |
| **有力** | 用神与辅神根深气足 | 身强煞露食神又旺，三者皆备 | 煞强食旺而身无根，或身强比重而财无气 |

### 格局高低判断

```
最高格：有情兼有力
  ↓
次高格：有情（但非至情）或有力（但非全力）
  ↓
低格：无情或无力
```

> 📖 原文："何格无贵？何格无贱？其理之大纲，亦在有情、有力无力之间而已。"

## 四、用神变化与纯杂

> 📖 原文第十、十一章

### 用神变化来源

1. **透干变化**：月令所藏不透本气而透其他（如寅月不透甲而透丙 → 用神变为丙）
2. **会合变化**：月令与别支三合三会（如寅月会午戌 → 木变为火）

### 变之美恶

- **变之而善**：辛生寅月逢丙，化财为官（忌变喜）
- **变之不善**：丁生亥月支全卯未，化官为印（官变印，失官之贵）
- **变而不失本格**：变出兼格，仍以月令本格为主

### 纯杂判断

| 类型 | 含义 | 示例 |
|------|------|------|
| **纯** | 互用而两相得 | 辛生寅月，甲丙并透，财官相生 |
| **杂** | 互用而两不相谋 | 壬生未月，乙己并透，官伤相克 |

> 📖 原文："纯者吉，杂者凶。互用而两相得者是也（纯）；互用而两不相谋者是也（杂）。"

## 五、用神配气候得失

> 📖 原文第十四章（与穷通宝鉴调候理论的衔接点）

- 格局用神与调候用神**不是同一概念**，但需互参
- 冬木逢水（印格），虽透官星亦难贵 → 金寒水冻，冻水不生木
- 伤官见官为祸，但金水伤官见官反为秀气 → 调候为急
- 夏木伤官佩印，其秀百倍 → 水火既济

> 📖 原文："论命惟以月令用神为主，然亦须配气候而互参之。"

## 六、相神紧要

> 📖 原文第十五章

### 用神与相神

```
用神 = 月令所取之主（君）
相神 = 辅佐用神成格之字（相）
```

| 格局 | 用神 | 相神 |
|------|------|------|
| 官逢财生 | 官 | 财 |
| 财旺生官 | 财 | 官 |
| 煞逢食制 | 煞 | 食 |
| 食神生财 | 食 | 财 |

### 关键规则

**"伤用神甚于伤身，伤相甚于伤用。"**

相神被破 = 格局立败。如：
- 甲用酉官，透丁（伤官），逢壬（合丁存官）→ 壬为相神，不可破
- 戊用子财，透甲（煞），逢己（合煞存财）→ 己为相神，不可破

## 七、四吉神能破格，四凶神能成格

> 📖 原文第十八、十九章

### 核心思想：名字 ≠ 好坏

| 吉神 | 滥用亦破格 | 凶神 | 得宜亦成格 |
|------|-----------|------|-----------|
| 财 | 食神带煞透财 → 财生煞为害 | 煞 | 印绶根轻透煞 → 煞生印为助 |

> 📖 原文："吉凶喜忌，全在是否合于需要，不因名称而有分别也。"

## 八、格局综合评价（等级体系）

> 📖 基于《子平真诠》"有情有力"核心框架，综合清澈浑浊、流转顺畅、调候得失、成格败格救应，评定格局等级。

### 评级维度（五维打分）

| 维度 | 权重 | 说明 |
|------|------|------|
| **格局清纯度** | 25% | 格局是否单一明确，有无混杂，有无破格之神 |
| **有情有力** | 25% | 用神与相神配合是否得当，是否根深气足 |
| **流转顺畅度** | 20% | 天干、地支、天地三层流转是否顺畅 |
| **调候得失** | 15% | 气候是否需要调候，调候用神是否得力 |
| **成格救应** | 15% | 是否成格，败格是否有救应 |

### 等级划分

#### 🟡 上等格局（上上、上中、上下）

**核心标准：** 格局清纯 + 有情有力 + 流转顺畅 + 调候得宜

| 细分 | 标准 |
|------|------|
| **上上格** | 五维皆为优。格局单一清纯，用神相神有情有力且通根透干，三层流转皆顺，调候恰得其用，成格无破。大富大贵之命，但极少见。 |
| **上中格** | 四优一良。格局清纯，有情有力，流转基本顺畅，某一层略有瑕疵但不影响大局。富贵可期，或某一方面特别突出。 |
| **上下格** | 三优两良，或有一维为中等但其他维度极强。格局基本清纯，有情有力，流转有小的阻塞但可通。中等富贵，或事业/某领域有显著成就。 |

**典型特征：**
- 格局单一明确（如纯正官格、纯食神格）
- 用神透出，相神有力，无第三者捣乱
- 天干流转连续相生，地支无严重冲刑
- 调候用神恰在要位
- 大运多走喜用之地

---

#### 🟠 中等格局（中上、中中、中下）

**核心标准：** 格局有一定条件，但某方面存在不足，需要大运配合

| 细分 | 标准 |
|------|------|
| **中上格** | 两优三良，或格局清纯但力量稍欠。格局明确，有情但非至情，或有力但非全力，流转基本顺畅。有成就，但需努力或待时机。 |
| **中中格** | 格局有一定混杂但有救应，或有情无力/有力无情，流转有明显阻塞但不至于全断。普通之命，一生平稳，小康或中等社会地位。 |
| **中下格** | 格局混杂较重，救应不力，无情或无力占其一，流转多处阻塞。一生多波折，需大运来救，成就有限。 |

**典型特征：**
- 格局基本成立但有小瑕疵（如财格透比劫但有食伤化劫）
- 用神有力但相神稍弱，或相神有力但用神位置不佳
- 天干或地支某一层流转阻塞
- 调候用神有但力量不足
- 大运好坏参半，需靠流年引动

---

#### 🔴 下等格局（下上、下中、下下）

**核心标准：** 格局破败，既无情又无力，流转三层皆阻

| 细分 | 标准 |
|------|------|
| **下上格** | 格局破败但有微弱救应，或大运某几步可走喜用。一生艰辛，但偶有转机，不至于绝境。 |
| **下中格** | 格局严重混杂无救，无情无力，流转阻塞严重。一生多困厄，需极大努力方能维持。 |
| **下下格** | 五维皆差。格局破败无救，既无情又无力，三层流转皆阻，调候失宜。极少见，需结合大运具体分析，不可轻言。 |

**典型特征：**
- 格局混杂严重（如官杀混杂、食伤混杂）
- 用神被严重克制或合化，相神无力
- 天干地支互战，无通关之神
- 调候用神缺失或被破
- 大运也多走忌神之地

### 评级操作流程

**Step 1：逐维评估**
对五个维度分别判断：优 / 良 / 中 / 差

**Step 2：综合定级**
根据五维分布对应到上述等级表中

**Step 3：给出理由**
每个等级必须附上具体理由，不可只给结论

**Step 4：标注大运影响**
说明当前大运对格局等级是"提升""维持"还是"拉低"。大运走喜用之地，中下格可升至中中；大运走忌神之地，上中格可能降至上下。

**重要原则：**
- 原局格局等级是"底子"，大运是"运气"。
- 原局上等格局，即使大运差也是"暂时不顺"；原局下等格局，即使大运好也是"昙花一现"。
- 评价格局等级时，以原局为主，大运为辅。可在大运分析中再详述大运对格局的动态影响。

## 使用边界

- 用神**必从月令取**，不可舍月令另寻
- 月令无用神时（建禄月劫），方看四柱另取
- 判断顺序：月令用神 → 成格败格 → 纯杂 → 有情有力 → 调候互参
- 星辰（天乙贵人等）无关格局，格局既成不因吉星而贵，格局既破不因吉星而救
- 外格（井栏、朝阳等）仅在月令确实无用神时才考虑
- 行运可以成格变格，原局之格非一成不变

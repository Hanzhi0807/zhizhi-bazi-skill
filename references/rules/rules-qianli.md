## 一、宫位取象体系

> 📖 原文"评断篇""六亲篇"

### 四柱宫位

| 柱 | 宫位 | 年龄段 | 主事 |
|----|------|--------|------|
| **年柱** | 祖上、父母、童年 | 1-16岁 | 祖荫、出身、根基 |
| **月柱** | 父母、兄弟、青年 | 17-32岁 | 学业、事业起点、社交圈 |
| **日柱** | 自身、配偶、中年 | 33-48岁 | 婚姻、核心事业 |
| **时柱** | 子女、晚年 | 49岁以后 | 归宿、子嗣、晚运 |

### 天干地支分工

- **天干**：明现之事，外在表现，公开的人际关系
- **地支**：暗藏之事，内在实质，家庭内部、内心感受
- **支中藏干**：隐伏之机，未显之缘

## 二、十神断事框架

> 📖 原文"六神篇""格局篇"

### 十神核心象义

| 十神 | 核心象义 | 宫位指向 |
|------|---------|---------|
| **正官** | 纪律、规则、上级、丈夫（女） | 年：祖上官职；月：上司、规章制度 |
| **七杀** | 压力、挑战、权威、魄力 | 时：晚年压力或权威地位 |
| **正印** | 学识、母亲、庇护、文书 | 月：学历；年：母亲 |
| **偏印** | 特殊技能、继母、玄学 | 月：另类才能 |
| **正财** | 工资、妻子（男）、稳定收入 | 日：配偶管理财 |
| **偏财** | 投资、父亲、意外之财 | 年：父亲；时：晚年财运 |
| **食神** | 才华、口福、悠闲、子息 | 时：子女；月：才艺 |
| **伤官** | 创造力、叛逆、口才、自由 | 月：职业倾向；时：晚年发挥 |
| **比肩** | 兄弟、朋友、同事、独立 | 月：朋友圈 |
| **劫财** | 竞争者、破财、果断 | 月：竞争对手 |

## 三、韦千里式断事方法

> 📖 原文"评断篇""应运篇"

### 核心流程

```
1. 看强弱（日主旺衰）
2. 定格局（月令取格）
3. 取用神
4. 论喜忌
5. 查岁运
6. 推六亲
7. 评性情
8. 断事业
```

### 韦千里的实战特点

1. **评语简明透澈，以少胜多**：不堆砌术语，一句话点中要害
2. **先从强弱下手**：强弱是定局之基，强弱不明，用神无从谈起
3. **用神不等于调候**：用神解决格局问题，调候解决季节问题，分开判断再合成
4. **重视"应期"**：原局断性质，岁运断发生时间

### 断事方法

| 步骤 | 方法 | 示例 |
|------|------|------|
| 看强弱 | 得令、得地、得助三要素 | "身强，喜财官" |
| 定格局 | 月令取格 + 格外变格 | "正官格，透财则贵" |
| 取用神 | 弱者扶，强者抑 | 身弱用印比，身强用财官食伤 |
| 论喜忌 | 扶用神者为喜，伤用神者为忌 | 用官者忌伤官 |
| 查岁运 | 大运为十年环境，流年为当年引动 | 岁运冲合用神为大事 |

## 四、历史校准方法

> 📖 此方法融合《千里命稿》实战精神与分析协议的校准流程

### 校准原则

1. **不定太细的时间**：说"18-22岁"，不说"19岁那年"
2. **必须给依据**：每个校准点都要说明从哪个柱子/十神/大运推导而来
3. **允许用户纠正**：用户反馈后修正用神/强弱假设
4. **禁止恐吓性断语**：用"可能有压力/转变"而非"必有灾祸"

### 校准事件模板

```text
校准点1（中置信度）：约在 XX-XX 岁，学业/工作/家庭 方面可能有 XX 转变。
推断依据：XX大运 + XX流年，引动了日柱/时柱的 XX 十神。

校准点2（中置信度）：约在 XX-XX 岁，XX 方面可能出现 XX。
推断依据：...
```

### 反馈修正

| 用户反馈 | 可能的问题 | 修正方向 |
|---------|-----------|---------|
| 校准点全部不对 | 用神判断可能有误 | 重新评估强弱/用神 |
| 部分对部分不对 | 某些柱子解释有偏差 | 调整十神映射 |
| 时间对事件不对 | 应期判断准确，象义解读需调整 | 调整十神取象 |
| 全部对上 | 基本判断可靠 | 可在此基础上给出最终建议 |

## 使用边界

- 韦千里的断语写于民国时期，涉及家庭角色、职业、婚恋等需转换到现代语境
- 书中某些硬性断语（如"女命伤官必克夫"）不可直接引用
- 历史校准的目的是**提高分析可靠性**，而非炫耀预测能力
- 若用户拒绝提供反馈校准，则跳过此步，直接给出带不确定性标注的分析

# 提示词示例

这些示例用于测试或演示 `zhizhi-bazi-skill`。

## 完整分析

```
使用 $zhizhi-bazi-skill。我的阳历生日是 1990 年 5 月 15 日，早上 8:30 出生，男，出生地上海。请先排盘，再分析日主旺衰、喜用神、当前大运，并在最终建议前给出 2 到 3 个历史事件校准点。
```

## 只知道大概方向

```
使用 $zhizhi-bazi-skill。请帮我看一个八字，重点看事业方向、学习方式和近几年大运流年。请先问我必要的出生信息，不要一次问太多问题。
```

## 时辰未知

```
使用 $zhizhi-bazi-skill。我只知道阳历生日是 1990 年 5 月 15 日，不知道具体出生时辰。请做一个不含时柱的部分分析，并明确说明哪些判断不能下结论。
```

## 夜子时边界

```
使用 $zhizhi-bazi-skill。出生日期是 1990 年 5 月 15 日，时间 23:30，女。请说明夜子时的采用约定，并指出哪些结论会受到这个约定影响。
```

## 审查已有解读

```
使用 $zhizhi-bazi-skill 审查一段八字解读。请检查它是否过度依赖单本经典、忽略调候旺衰、使用宿命化语言，或者跳过了历史事件校准。
```
